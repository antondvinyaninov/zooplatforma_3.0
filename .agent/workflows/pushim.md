description: Завершить работу над модулем, слить готовый код в main и запушить релиз
---
При вызове команды `/pushim` или фразе "Пушим модуль", вы должны выполнить следующие шаги:

// turbo-all
1. Закоммитьте финальный код в ветке модуля: `git add . && git commit -m "feat: complete module"` (если изменений нет, игнорируйте).
2. Выясните имя текущей ветки перед переключением (чтобы знать, что мы сливаем): `git branch --show-current`
3. Переключитесь на главную ветку: `git checkout main`
4. Подтяните обновления (чтобы избежать конфликтов): `git pull origin main`
5. Соедините наш модуль с основной веткой: `git merge [сохраненное имя_ветки]`
6. Отправьте итоговый код на GitHub: `git push origin main`
7. Сообщите пользователю, что модуль успешно выкачен в продакшен, и при желании он может вызвать команду `/zapishi` для внесения фичи в `CHANGELOG`.
