---
description: Приостановить работу над текущим модулем и переключиться в main для срочного исправления ошибки
---
При вызове команды `/pravim` или фразы "Правим главную", вы должны выполнить следующие шаги:

// turbo-all
1. Закоммитьте все текущие черновые изменения в ветку модуля: `git add . && git commit -m "wip: pause module for urgent hotfix"` (если изменений нет, игнорируйте ошибку команды).
2. Безопасно переключитесь на ветку main: `git checkout main`
3. Подтяните все актуальные обновления: `git pull origin main`
4. Выведите пользователю сообщение: "Ветка `main` активирована. Черновик модуля сохранен. Кодовая база чиста, я готов чинить баг!".
